## VIOLATION #1: Article I, Section 10 - Contracts Clause

### The Constitutional Text:

> **U.S. Constitution, Article I, Section 10, Clause 1:**
> "No State shall... pass any Law impairing the Obligation of Contracts..."

### What It Means:

States cannot pass laws that impair (interfere with, weaken, damage) the obligation of contracts.

### How Foreclosure Violates It:

#### The Logic:

```
1. Valid contract requires CONSIDERATION
   ↓
2. Bank gave NO consideration (created entries from borrower's signature)
   ↓
3. No consideration = VOID contract (Black's Law Dictionary, p. 1902)
   ↓
4. Void contract has NO OBLIGATION (void = "of no legal effect")
   ↓
5. State foreclosure laws attempt to ENFORCE void contract
   ↓
6. Enforcing void contract = CREATING obligation where none exists
   ↓
7. This IMPAIRS the constitutional requirement that contracts be valid
   ↓
8. VIOLATES Article I, Section 10 (Contracts Clause)
```

#### The Argument:

**Traditional Understanding:**
- Contracts Clause protects existing contract obligations from state interference

**Deeper Understanding:**
- Contracts Clause also protects constitutional requirement that contracts be VALID
- Valid contract requires consideration (fundamental contract law)
- State cannot use power to enforce void contracts as if they have legal effect
- This impairs the OBLIGATION that contracts must be valid to be enforceable

**Application:**
```
State uses foreclosure laws to enforce contract void for lack of consideration.

State is attempting to:
1. Create obligation where none exists (void contract = no obligation)
2. Give legal effect to void contract (void = "no legal effect")
3. Enforce invalid contract as if valid
```
