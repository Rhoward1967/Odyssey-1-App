## 🔧 DATABASE INTEGRATION

### **SQL View for Toll Booth Tracking**

```sql
CREATE OR REPLACE VIEW toll_booth_monthly_summary AS
SELECT
  DATE_TRUNC('month', invoice_date) AS month,
  SUM(amount) AS gross_revenue,
  SUM(amount * 0.05) AS admin_fee_5pct,
  SUM(amount * 0.95 * 0.35) AS mastery_tax_35pct,
  SUM(amount * 0.95 * 0.65) AS net_operations_60pct
FROM invoices
WHERE company = 'HJS Services LLC'
  AND status = 'paid'
GROUP BY DATE_TRUNC('month', invoice_date)
ORDER BY month DESC;
```

### **Query Current Month:**

```sql
SELECT * FROM toll_booth_monthly_summary
WHERE month = DATE_TRUNC('month', CURRENT_DATE);
```
