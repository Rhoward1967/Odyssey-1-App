## STEP 9: DOWNLOAD AS PDF

1. In Google Docs, click **File** → **Download** → **PDF Document**
2. Name: **`ODYSSEY-1_TRUST_DOCUMENTS_20260207.pdf`**
3. Save to Desktop or Documents folder
4. **Do NOT print from Google Docs** - Save as PDF first, then print from PDF

**Why PDF instead of Google Docs print?**
- PDF preserves all formatting exactly as intended
- Printer shop can open PDF on their system
- No font substitution issues
- Signature pages print clearly (no ghosting)
- Notary seal spaces remain empty and clear

## TROUBLESHOOTING COMMON ISSUES

**Problem:** Text looks blurry after pasting into Google Docs
- **Solution:** This is normal. Google Docs sometimes renders markdown differently. The downloaded PDF will be clear. Proceed with download.

**Problem:** Page breaks look weird in Google Docs
- **Solution:** Normal. PDF download will fix layout. Don't worry about Google Docs appearance.

**Problem:** Signature lines look too short
- **Solution:** They're fine. Printer will preserve line length. Test on sample page if concerned.

**Problem:** Notary seal spaces not clearly visible
- **Solution:** Check PDF. They should have "[NOTARY SEAL SPACE]" or similar text. Tell printer to leave that area completely blank.

**Problem:** Can't open PDF file
- **Solution:** Make sure PDF reader is installed. Try Adobe Acrobat Reader (free). Google Drive can preview PDFs too.
